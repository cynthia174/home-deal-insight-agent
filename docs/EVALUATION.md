# 准确率评测说明

| 检查项 | 结果 |

## 覆盖范围

24 道题覆盖：

- 总签约额、设计费、毛利额；
- 项目数、平均值、客单价；
- 年份和月份筛选；
- 城市、风格、渠道、房屋类型、设计师；
- Top N 和 Bottom N 排名；
- 月度趋势图和分类柱状图；
- 毛利率、按时出图率；
- 已取消合同口径；
- 多条件组合。

## 如何运行

在产品页面底部点击“运行完整准确率评测”，或在项目目录运行：

```bash
python evaluation/evaluate.py
```

全部通过时命令返回退出码 0；任何题目失败都会返回退出码 1，适合接入持续集成。

## 黄金答案如何产生

`evaluation/build_golden_set.py` 使用独立的 Pandas 口径计算正确答案，并把数值固化到 `golden_questions.json`。日常评测只读取固化答案，不会随着 Agent 的代码变化而自动改变。

如果演示数据本身发生有意修改，需要重新生成黄金答案：

```bash
python evaluation/build_golden_set.py
python evaluation/evaluate.py
```

## 判定标准

一题通过需要同时满足：

- Agent 没有运行错误；
- 指标识别正确；
- 分组维度识别正确；
- 图表题的图表类型正确；
- 实际结果和黄金答案的行数、分类、排序、数值全部一致；
- 浮点数满足 `rel_tol=1e-8`、`abs_tol=1e-6`。
