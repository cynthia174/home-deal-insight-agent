# 小白使用手册

## 一、怎么打开

Windows 用户只需要双击项目根目录的 `run_app.bat`。

第一次打开时，电脑会自动准备运行环境并安装依赖，通常需要几分钟。看到浏览器中的“家装经营洞察 Agent”页面后，就可以关闭黑色命令窗口以外的其他操作了；使用期间不要关闭这个命令窗口。

如果浏览器没有自动打开，请在浏览器地址栏输入：

```text
http://localhost:8501
```

如果 8501 已被其他程序使用，页面会在命令窗口显示新的可用地址。

## 二、怎么提问

页面默认使用演示数据。可以点击页面上的示例问题，也可以在底部输入框中输入自己的问题，然后按回车。

推荐从这三题开始：

1. `去年总签约额是多少？`
2. `签约额最高的前三种设计风格是什么？`
3. `帮我画一张每个月签约额变化图`

每次回答会包含：

- 一句直接结论；
- 计算口径说明；
- Agent 的完整工作步骤；
- 计算结果明细；
- 需要时自动生成的图表。

## 三、怎么展示自动纠错

1. 在左侧打开“演示一次自动纠错”。
2. 点击“去年总签约额是多少？”。
3. 展开“发现错误并自动修复”步骤。
4. 页面会显示第一次遇到的字段错误、修复动作和重新执行后的正确答案。

这个开关只用于直观展示恢复过程。即使不开启，上传文件时的字段别名、金额格式、日期格式和异常值修复仍会正常工作。

## 四、怎么上传自己的表

1. 左侧选择“上传自己的表”。
2. 点击上传区域，选择 `.xlsx`、`.xls`、`.csv` 或 `.tsv` 文件。
3. 系统检查字段并显示可提问页面。

最少需要这些信息：

| 标准字段 | 可自动识别的常见别名 |
| --- | --- |
| 项目编号 | 项目ID、订单编号、合同编号 |
| 签约日期 | 签单日期、合同日期、成交日期 |
| 签约额 | 合同金额、签单金额、成交金额、销售额 |
| 设计风格 | 装修风格、风格 |
| 合同状态 | 合同阶段、订单状态 |

如果有预计成本，缺少毛利额时系统会自动按“签约额 - 预计成本”补算。

## 五、怎么运行准确率评测

页面底部展开“产品能力验证｜24 道黄金题自动评测”，点击“运行完整准确率评测”。

评测会逐题运行，并展示通过题数、准确率和每道题结果。也可以在项目目录的 PowerShell 中运行：

```powershell
python evaluation/evaluate.py
```

## 六、常见问题

### 页面打不开

确认启动程序的黑色窗口没有被关闭。重新双击 `run_app.bat` 即可。

### 上传后提示缺少字段

打开 Excel 第一行，确认它是字段名而不是大标题，并参考上面的必需字段表。系统会自动识别常见别名。

### 问题没有得到预期答案

尽量在一句话里写清楚指标、时间和维度，例如：

```text
2025年各城市的平均签约额
```

比“帮我看看去年表现”更容易得到精确、可核验的结果。

### 业务数据会上传到网络吗

不会。当前版本的核心分析和计算全部在本机完成。
